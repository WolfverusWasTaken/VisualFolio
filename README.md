# VisualFolio

My personal portfolio site. Built with React + Vite.

## Running locally

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173/VisualFolio/`

## Deploying

Push to `main` branch - GitHub Actions handles the rest. Site goes live at:
https://WolfverusWasTaken.github.io/VisualFolio/

## Stack

- React
- Vite
- Tailwind CSS
